# 3D Dice Simulator

Roll the dice!

Built using vite + react + tailwindcss + babylon.js

## How to Use

```bash
# Install dependencies
yarn

# Run dev server
yarn dev

# Build for production
yarn build

# Preview production build locally
yarn preview

# Lint files
yarn lint

# Publish build to github pages (note: run build command first)
yarn run publish
```
